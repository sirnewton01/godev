orion-codemirror

This is a plugin for Orion that provides syntax highlighting using CodeMirror modes. See below for installation instructions.

Requirements

  • Orion version 4.0M1, or lower.

Installation

  1. Copy the following URL:
  2. Log in to your Orion environment, open the user menu, and click the "Settings" link.
  3. Select the "Plugins" category, click "Install", paste the URL into the field, and click "Submit".

Usage

  1. In Orion, create a new file with an extension corresponding to one of the modes listed below. (For example, test.php is associated with the php mode). Click the file.
  2. An editor will open. Type some (valid) code in the editor, and it will be syntax-colored.
